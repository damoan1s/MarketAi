# MX – Market Intelligence Layer

MX هو الطبقة العقلية العليا داخل مشروع MarketAI.  
يمثل مركز التحليل المؤسسي الحقيقي حيث يتم دمج:

- النماذج الذكية  
- قواعد النية  
- تحليل الفيوتشر والفوري  
- البنية الهندسية للسوق  
- نماذج الامتصاص  
- نماذج عدم تناسق الذيول  
- نماذج قياس الفروقات الحركية  

MX هو الذي يحدد:
- من يقود السوق (Futures أو Spot)
- هل الحركة مصطنعة أو حقيقية
- هل السيولة تُصاد
- هل الاتجاه سينعكس
- هل القوة حقيقية أم مزيفة

الطبقة MX هي العمود الفقري لنظام DNA Engine.
