# Wick Asymmetry Rules

1. إذا كان ذيل الفيوتشر أكبر → امتصاص سيولة عليا (Sell Intent).
2. إذا كان ذيل السبوت أكبر → امتصاص سيولة سفلية (Buy Intent).
3. إذا ظهرت 3 شموع في الفيوتشر تبتلع ذيولها → Fake Breakout.
4. اختلاف الذيول يكشف الخداع قبل الاتجاه.
