# WORK PROTOCOL — AI Team Collaboration System
Version: 1.0  
Prepared for: Anis + ChatGPT + Claude + DeepSeek  
-------------------------------------------------------------

# 1. الهدف من البروتوكول
هذا البروتوكول يحدد طريقة العمل الرسمية بين:
- أنيس (Chief Architect)
- ChatGPT (Systems Engineer)
- Claude (Documentation AI)
- DeepSeek (Performance & Debugging Engine)

الهدف:
تنسيق العمل — منع العشوائية — توحيد طريقة التفكير — تسريع التطوير.

-------------------------------------------------------------

# 2. متى يُستخدم كل نموذج؟

## 2.1. ChatGPT (Systems Engineer)
يتم استدعاؤه عندما تحتاج إلى:
- بناء ملفات المشروع الداخلية  
- تنظيم المجلدات  
- كتابة الأكواد الهندسية  
- تطوير الـ ROOT1 → ROOT4  
- بناء decision engines  
- دمج الفلسفة مع النظام  
- مراجعة اتساق الملفات  
- التفكير المنطقي العميق  

❗ ChatGPT هو نموذج الهندسة والتسلسل المنطقي.

---

## 2.2. Claude (Documentation & Philosophy Developer)
يتم استدعاؤه عندما تحتاج إلى:
- كتابة شروحات طويلة  
- كتابة فلسفة المشروع  
- كتابة ملفات ARCHITECTURE.md و DNA.md  
- توسيع أفكارك  
- صياغة النظريات بشكل احترافي  
- تحسين النصوص  
- إنشاء وثائق للمستثمرين أو المؤسسات  

❗ Claude هو نموذج الكلام الطويل والفلسفة والوثائق الرسمية.

---

## 2.3. DeepSeek (Optimizer / Debugger)
يُستدعى عندما تحتاج:
- كتابة أكواد low-level  
- تسريع backtesting  
- تحسين الأداء  
- تصحيح الأخطاء  
- refactor للكود  
- تحسين memory usage  
- معالجة مشاكل الرسم أو المؤشرات  

❗ DeepSeek هو نموذج الحلول التقنية الثقيلة.

-------------------------------------------------------------

# 3. منهجية العمل اليومية (Daily Pipeline)

## 3.1. بداية اليوم
1. مراجعة:
   - MEMORY_BOARD.md  
   - ROADMAP_UPDATE_PROTOCOL.md  
2. تحديد هدف واحد فقط لليوم.  
3. فتح مجلد progress/dayX.  
4. إنشاء:
   - summary.md  
   - notes.md  
   - README.md  
   - card.py  
   - test.py  

## 3.2. أثناء العمل
تطبق قاعدة 3 Levels:
1. Thinking → مع أنيس  
2. Structuring → مع ChatGPT  
3. Documenting → Claude  
4. Optimizing → DeepSeek  

## 3.3. نهاية اليوم
- تحديث Memory Board  
- تحديث Roadmap  
- تحديد اليوم التالي  
- أرشفة يومك داخل progress/dayX  

-------------------------------------------------------------
